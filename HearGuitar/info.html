Welcome to HearGuitar

Welcome! This app is designed to help you learn guitar using sound and step-by-step guidance.

Controls

  • Chord Teacher Arrow Functions:
  • Right Arrow: Next instruction
  • Left Arrow: Previous instruction
  • Down Arrow: Repeat instruction (voice only)
  • Up Arrow: Repeat sound

  • Song Teacher Arrow Functions:
  • Right Arrow: Next chord
  • Left Arrow: Previous chord
  • Down Arrow: Repeat chord (voice only)
  • Up Arrow: Play chord sound

Guitar Basics

Fretboard

The guitar has six strings. The first string is the thinnest (high E), and the sixth string is the thickest (low E).

Frets are the metal bars along the neck. Pressing a string on a fret changes the pitch.

Every move up the fretboard on the same string increases the pitch by a half note.

Bar

A bar means using one finger to press down every string across a fret. This is done with your index finger.

Mute

Muting a string means you either touch it lightly so it does not produce a sound, or you forgo strumming it.
